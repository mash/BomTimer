Bom Timer
====

Overview
時間の経過に応じてLEDの光り方が変わるArduinoのスケッチです。


## Description
スイッチを押すと、タイマーが起動し時間の経過にともなってLEDの光り方がだんだん激しくなっていきます。指定した時間に達するとタイマーは止まります。

## Usage
`sketch.ino`をArduinoで開いてご使用ください。
